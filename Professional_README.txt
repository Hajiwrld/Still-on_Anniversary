PREMIUM ANNIVERSARY WEBSITE

Project

A romantic one-page website created as a digital love letter. It is
designed to be published with GitHub Pages and shared using a public
HTTPS link.

Project Files

index.html - Main website file README.txt - Project documentation

Features

-   Elegant blue gradient background
-   Glassmorphism love letter card
-   Animated floating hearts
-   Falling rose animations
-   Typewriter text effect
-   Mobile-friendly responsive layout
-   Romantic “Open My Heart” button

How to Publish on GitHub Pages

1.  Sign in to GitHub.
2.  Create a new repository.
3.  Upload index.html to the repository.
4.  Commit the changes.
5.  Open: Settings -> Pages
6.  Under “Build and deployment”: Source : Deploy from a branch Branch :
    main Folder : /(root)
7.  Click Save.
8.  Wait 1-3 minutes.

Your website will be available at:

https://YOUR-USERNAME.github.io/REPOSITORY-NAME/

Customization

You can edit the text inside index.html to: - Change names - Change
colors - Add photos - Add music - Update the love letter

Browser Support

Works in modern versions of: - Chrome - Edge - Firefox - Safari
